# Mode: surgical-edit

Minimal change. Preserve wording and structure everywhere else.

## Steps

1. Identify the exact span and the user constraint (comment, typo, one claim).
2. Apply evidence and conflict gates **only** to claims you will change.
3. Do **not** load full section hourglasses or run a full revision-pass stack.
4. Load [scaffolding.md](../scaffolding.md) only if the edit touches Methods
   or Results prose that might introduce scaffolding.
5. Load the relevant fragment of [reference.md](../reference.md) only if the
   edit needs a section rule (e.g. citation placement, statistical honesty).
6. Change the fewest words that satisfy the request.
7. Skip the storyboard and skip whole-manuscript re-argument.

**Done when** the requested change is made, adjacent wording is preserved,
no unsolicited rewrite of other paragraphs occurred, and any `[TBC]` or
conflict created by the edit is reported.

## Output

1. The edited span (or minimal surrounding context)
2. Material `[TBC]` / conflicts if the edit surfaces them
3. No essay on what you chose not to change
